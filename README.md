# Embedded Systems Archive

> A personal archive for learning embedded systems.

## Languages

This repository contains my study notes, example code, hardware projects, and experiments related to embedded systems.

The goal of this archive is to organize what I learn while building embedded software and hardware, making it easier to review concepts and track my progress over time.
